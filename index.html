01. 소개

Stony Brook University에서 기술경영학을 전공하며 기술과 사회의 상호작용에 대한 깊은 이해를 쌓았습니다. 학부 시절부터 인공지능과 임베디드 시스템의 잠재력에 매료되어, 두 분야의 융합을 통해 실생활의 문제를 해결하는 데 집중해왔습니다.

저의 가장 큰 강점은 소프트웨어 알고리즘부터 하드웨어 레벨의 최적화까지 아우르는 폭넓은 기술 스펙트럼입니다. 최신 AI 모델을 경량화하여 저전력 임베디드 보드에 탑재하거나, RTOS 환경에서 실시간 데이터 처리를 위한 효율적인 시스템을 설계하는 등 복합적인 문제 해결에 능숙합니다.

새로운 기술을 배우고 적용하는 것을 즐기며, 동료들과의 협업을 통해 더 나은 결과물을 만들어낼 때 큰 보람을 느낍니다. 저의 기술과 경험이 어떻게 새로운 가치를 창출할 수 있을지 항상 기대하고 있습니다.

03. 프로젝트

Jetson Nano에서 실시간으로 작동하는 YOLOv5 모델을 최적화하고 배포했습니다. 저전력 환경에서 높은 프레임률을 달성하여 실시간 감시 시스템에 적용했습니다.

Python PyTorch TensorRT Jetson Nano

C++ ESP32 FreeRTOS MQTT PlatformIO

Python TensorFlow Reinforcement Learning AirSim
